# Pariksha_Nirikshak
A smart surveillance system that uses AI and computer vision to ensure fair examinations.

Pariksha Nirikshak is an AI-powered proctoring system designed to monitor offline examinations and detect suspicious activities in real time. Using Computer Vision and Machine Learning, the system analyzes live CCTV or webcam footage to identify behaviors such as frequent head movements, looking away from the answer sheet, and other potential cheating patterns.

Objective: The goal of Pariksha Nirikshak is to assist invigilators by providing an intelligent monitoring solution that enhances exam integrity and reduces manual supervision efforts.

🚀 Features:
Real-time exam monitoring.
AI-based suspicious behavior detection.
Computer vision for face and head movement tracking.
Automated alerts for potential cheating attempts.
CCTV/Webcam integration.
Scalable for classrooms and examination halls.

🛠️ Technologies Used:
Python
OpenCV
Machine Learning
Computer Vision
NumPy
CCTV/Webcam Video Processing
