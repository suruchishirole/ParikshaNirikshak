# Pariksha_Nirikshak
<i>A smart surveillance system that uses AI and computer vision to ensure fair examinations.</i>

<b>Pariksha Nirikshak</b> is an AI-powered proctoring system designed to monitor offline examinations and detect suspicious activities in real time. Using <b>Computer Vision and Machine Learning</b>, the system analyzes live CCTV or webcam footage to identify behaviors such as frequent head movements, looking away from the answer sheet, and other potential cheating patterns.

Objective: The goal of Pariksha Nirikshak is to assist invigilators by providing an intelligent monitoring solution that enhances exam integrity and reduces manual supervision efforts.

🚀 Features:
<br>Real-time exam monitoring.
<br>AI-based suspicious behavior detection.
<br>Computer vision for face and head movement tracking.
<br>Automated alerts for potential cheating attempts.
<br>CCTV/Webcam integration.
<br>Scalable for classrooms and examination halls.

🛠️ Technologies Used:
<br>Python
<br>OpenCV
<br>Machine Learning
<br>Computer Vision
<br>NumPy
<br>CCTV/Webcam Video Processing
